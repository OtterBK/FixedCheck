



## ? FixedCheck, [ 고정 수표 플러그인 ]
> 마인크래프트 서버 플러그인 (1.12.2 버전)
  
---  

#### ? 어떤 플러그인인가요?
마인크래프트에 커스텀 NPC 모드에서 상점 시스템을 사용하기 쉽도록 정하진 가격만큼의 수표를 생성하는 플러그인입니다.

#### ? 명령어
```  
* /수표 - 수표 관련 명령어 목록을 표시합니다.
```
